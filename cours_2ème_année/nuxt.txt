NUXT
opensource basé sur le language VUE (en 2017)
routing auto basé sur la nomanclature et l'architecture des fichiers.
si on commence le nom du fichier par :  cela devient un parametre.
serveur side randoring (SSR) = genere html coté serveur avant d'envoyer coté client, donc meilleur performance, meilleur referencement, meilleur experience utilisateur.
possible de desactivé le ssr pour avoir des données assyncrone.
version 3 de nuxt = type script.



INSTALLATION:
npm install -g create-nuxt-app     (pour avoir juste quelque composant sinon faire = npm i nuxt)

CREER UN PROJET:
npx create-nuxt-app exercice2bci   
(pour configurer l'app, espace pour selectionner les choix multiples puis entrer pour valider)
